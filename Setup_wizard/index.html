Water Level Project Setup

Follow the steps below to configure your system.

Step 0: Install Docker
Ensure Docker is running before generating your .env file: Windows/Mac | Linux

1. Environment

Configure ports, passwords, and database settings for Docker.

Open .env Tool

2. Receiver Firmware

Generate firmware for the T-Beam Receiver with SSL certificates.

Open Receiver Tool

3. Sender Firmware

Generate firmware for the Sender nodes (e.g., MK1, MK2).

Open Sender Tool

Place all generated files in: /WaterLevelProject-main
