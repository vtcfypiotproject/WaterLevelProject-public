.env Configuration Setup

Installation Requirements:
  1. Download and install Docker: Docker Desktop (Windows/Mac) or Docker Engine (Linux).
  2. Crucial: Please start the docker service.
  3. Use the following command in the powershell to get your Node-RED hashed password
    • docker run -d -p 1880:1880 --name mynodered nodered/node-red:4.1.1-22
    • docker exec -it mynodered node-red admin hash-pw
    • docker rm -f mynodered
  4. Fill in all the fields below.
  5. Crucial: Move the downloaded file to the root directory of your /WaterLevelProject-main folder.
  6. Rename you file as ".env"
